## 4. 가중 진척률 계산 (참고용)

각 Sprint를 단순 평균하면:

```
(100 + 100 + 85 + 100 + 95 + 85 + 55 + 골격) / 7 ≈ 88.6%  (Sprint 7은 가중 0.5로 포함 시)
```

각 Pillar를 단순 평균하면 (Sprint 7 반영):

```
(85 + 100 + 100 + 95 + 60 + 80 + 85) / 7 ≈ 86.4%
```

종합적으로 **약 85~87% 진척** (↑ Sprint 7 골격 선도입 효과), 잔여 15%는 **외부 벤더 연동(Qdrant sparse·Reranker·OTel collector·OAuth2 IdP)**, **Pillar 5 DAG화**, **a11y·chaos 6종** 으로 명확히 식별된다.
