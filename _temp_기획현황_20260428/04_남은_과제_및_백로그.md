## 6. 다음 한 사이클(2주)에 권장하는 묶음

> "외부 의존이 없는 내부 작업"과 "외부 의존이 있는 작업"을 분리해 병렬 진행하는 것을 권장.

**Track A (내부, 1인으로 즉시 가능)**
- 노드 내부 true streaming 리팩터링
- Adversarial 30종 케이스 작성 + 차단율 측정 셋
- chaos test 6종 시나리오 작성

**Track B (외부 의존, 선결정 필요)**
- 진짜 RAGAS 패키지 도입 + CI 임계 재조정
- OpenTelemetry collector 선택 + 분산 트레이싱 연결
- OAuth2 IdP 선택 + 인증 모드 추가

> 두 트랙이 모두 닫히면 **DoD 7항목 중 잔여 1.5건이 그린**으로 들어와 "현대 최고 수준" 선언이 가능해진다.
