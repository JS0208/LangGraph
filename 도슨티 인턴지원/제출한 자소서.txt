지원동기
학부에서 AI소프트웨어를 전공하며 늘 가졌던 고민은 ‘내가 짠 코드가 실제 비즈니스 환경의 복잡
한 변수를 견딜 수 있을까’였습니다. 정제된 데이터로 모델을 돌리는 학교 프로젝트와 달리, 실무에
서는 예측 불가능한 에러와 노이즈가 끊임없이 발생한다고 생각합니다. 저는 단순히 모델 정확도
를 높이는 것을 넘어, 이런 변수를 통제하고 서비스가 멈추지 않도록 백엔드 파이프라인을 견고하
게 설계하는 과정에 큰 흥미를 느낍니다.
도슨티에 지원한 이유는 제가 치열하게 부딪히며 설계해 본 아키텍처를 실제 프로덕트 환경에 투입
해 성과를 내고 싶기 때문입니다. 특히 고객사마다 다른 형태의 비정형 데이터를 RAG로 엮어내
는 도슨티의 맞춤형 챗봇 서비스는, 제가 고민해 온 데이터 처리 및 예외 상황 제어 역량을 가
장 잘 발휘할 수 있는 분야라고 확신합니다.
단순히 배우고 가는 실습생에 머물지 않겠습니다. 실제 유저 트래픽과 로그 데이터를 마주하며 파
이프라인의 병목을 주도적으로 찾아내고, 이를 코드로 개선하여 회사의 서비스 고도화에 즉각적으
로 기여하는 주니어 엔지니어의 태도로 임하겠습니다.

본인의 강점 및 발휘경험
저의 핵심 역량은 여러 기술을 엮어 실제 동작하는 파이프라인을 혼자서 설계하고 구현해 내는 '실
행력'입니다.
최근 단일 LLM의 환각 현상을 제어하기 위해 LangGraph 기반의 재무 분석 시스템을 개발했습니
다. 전체 과정을 모델 하나에 통째로 맡기면 에러 추적이 어렵다는 것을 깨닫고, 역할을 8개의 에이
전트로 쪼개어 자가 교정 루프를 구축했습니다. 또한 정보 간의 복잡한 관계를 정확히 찾기 위해 단
순 벡터 검색(Qdrant) 대신 지식 그래프(Neo4j)를 결합한 고도화된 RAG 검색을 구현해 FastAPI
로 서빙했습니다. 이 과정에서 프록시 지연 등 실제 장애 시나리오를 미리 가정하고 예외 처리 로직
을 단단하게 구성했습니다.
이와 함께 대규모 통계 모델링 프로젝트도 진행 중입니다. 인간의 비합리적 의사결정을 분석하
기 위해 수만 건의 행동 데이터를 직접 전처리하고, 기존 확률 모델과 양자 인지 모델 간의 오차
를 검증하는 파이프라인을 다루며 데이터 분석의 깊이를 더하고 있습니다.
제가 코드를 짤 때 가장 타협하지 않는 부분은 '교차 검증'입니다. 결과가 한 번 잘 나왔다고 맹신하
지 않고, 예외 케이스를 집요하게 테스트하여 데이터 정합성을 확인하는 원칙을 실무에서도 지키겠
습니다.

역량기술서
실무에 즉시 투입될 수 있도록 다음과 같이 기술 스택을 다지고 결과물을 만들어 왔습니다.
[1] 재무 의사결정 지원 시스템(DSS) 개발
LangGraph를 활용한 8개 전문화 멀티 에이전트 설계 및 환각 제어 루프 구축
복잡한 데이터 관계 추적을 위해 그래프 DB(Neo4j)와 벡터 DB(Qdrant)를 결합한 하이브리드 기
반 RAG 검색 구현
FastAPI를 이용한 백엔드 서빙 및 비동기 태스크 기반 장애 예외 처리 로직 적용
[2] 대규모 데이터 기반 의사결정 패턴 분석
인간의 비합리적 선택을 모델링하는 양자 인지 프레임워크 기반 복잡계 연구
Many Labs 2 등 수만 건의 행동 데이터를 전처리하고 통계 모델 예측 오차 교차 검증
[3] 맞춤형 마라톤 페이스 예측 서비스 'FitPace'

데이터 전처리부터 모델 학습, Streamlit 기반 웹 서비스 배포까지 전체 사이클 경험
