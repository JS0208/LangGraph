## ✏️ 본문

LLM 기반 멀티에이전트 시스템을 처음부터 혼자 설계하고 운영했습니다.

숭실대학교 컴퓨터공학부 재학 중, GraphRAG 기반 금융 분석 멀티에이전트 시스템을 개발했습니다. LangGraph로 의도 분류·컨텍스트 검색·재무 분석·리스크 감사·반성·비평 등 8개 전문화 에이전트를 오케스트레이션하고, FastAPI RESTful API와 마이크로서비스 패턴으로 서비스 레이어를 구성했습니다. 그래프 DB(Neo4j)와 벡터 DB(Qdrant)를 결합한 하이브리드 검색으로 컨텍스트 정확도를 높였습니다.

이 프로젝트의 핵심 과제는 '실시간성'이었습니다. 에이전트 파이프라인 처리 결과를 지연 없이 클라이언트에 전달하기 위해 SSE(Server-Sent Events)와 asyncio 비동기 큐를 직접 설계했습니다. 구현 중 네트워크 유휴 타임아웃으로 스트림이 조용히 종료되는 장애를 마주했습니다. 가능한 실패 시나리오를 먼저 열거하고 아키텍처를 역설계하는 방식으로 접근했습니다. ping 이벤트 주기 전송으로 연결을 유지하고, 클라이언트 이탈 시 asyncio 태스크 누수를 cancel 인터럽트 핸들러로 차단해 안정성을 확보했습니다.

모니터링과 보안 계층도 직접 구성했습니다. Prometheus 메트릭과 OpenTelemetry 분산 추적으로 노드별 지연·오류를 가시화하고, JWT 기반 인증과 감사 로그로 글로벌 서비스 보안 요건을 염두에 둔 설계를 적용했습니다.

커넥티드 카 백엔드가 요구하는 실시간 알림 시스템 고도화와 지능형 모니터링은 이 경험에서 직접 파생됩니다. 컴퓨터공학 전공과 금융학부 부전공을 교차하며 기술과 도메인을 함께 보는 시각을 갖고 있습니다. 절대적인 정답보다 현재 최선의 선택을 실행하고 검증하며 수정하는 방식으로, 팀과 함께 더 나은 시스템을 만들어가겠습니다.
